# NotebookLm_MenteBrilhante_IA
ste projeto utiliza o NotebookLM para criar uma persona de consultoria de alto nível, "treinada" exclusivamente em papers seminais, manifestos técnicos e visões de futuro dos líderes da IA (como Andrej Karpathy, Alan Turing e pesquisadores do Google DeepMind). O objetivo é unir o rigor técnico à inspiração visionária.
